# NotaryCafe Agent Plugin

Automates NotaryCafe.com — login, search for notaries, scrape profiles, and send messages.

## Tools

- `nc_login` — Log into NotaryCafe.com with email and password
- `nc_search` — Search for notaries by zip code or city
- `nc_scrape_profile` — View a notary's profile details
- `nc_send_message` — Send a message to a notary via their profile

## Usage

Message OpenClaw via Telegram and use natural language like:
- "Login to NotaryCafe as my@email.com with password mypass"
- "Search for notaries in 97201"
- "Show me profile at /agent.name"
- "Send a message to profile /agent.name saying: Hi, I need a notary"
