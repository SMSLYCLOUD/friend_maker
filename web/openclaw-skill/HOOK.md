# IG Agent Plugin

Instagram automation plugin for OpenClaw — provides tools for login, follow, unfollow, send DMs, scrape followers, and run bombing campaigns.

## Tools

- `ig_login` — Log into Instagram with username/password or session cookies
- `ig_follow` — Follow an Instagram user
- `ig_unfollow` — Unfollow an Instagram user
- `ig_send_dm` — Send a DM to an Instagram user
- `ig_scrape_followers` — Scrape followers from any public account
- `ig_bombing_campaign` — Run mass follow or mass DM campaigns

## Usage

Message OpenClaw via Telegram and use natural language commands like:
- "Login to Instagram as @myusername with password mypassword"
- "Follow @targetuser"
- "Send a DM to @someone saying: Hey! Let's connect."
- "Scrape 100 followers from @competitor"
- "Run a mass follow campaign on @target followers, count 50"
